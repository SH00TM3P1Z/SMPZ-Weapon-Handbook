# DayZ 총기 도감 사이트

DayZ 총기 모드의 총기 정보를 관리하는 웹사이트입니다.

## 기능

- 좌측 사이드바에 카테고리 목록 표시
- 각 카테고리별 총기 수 표시
- 카테고리 클릭 시 해당 카테고리의 총기 목록 표시
- 관리자만 무기 정보 등록/수정/삭제 가능

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 좌측 사이드바에서 원하는 카테고리 클릭
3. 관리자 로그인:
   - 기본 관리자 계정: `admin` / `admin123`
   - 로그인 후 무기 추가/수정/삭제 가능

## 파일 구조

- `index.html` - 메인 HTML 파일
- `styles.css` - 스타일시트
- `script.js` - JavaScript 로직
- `README.md` - 이 파일

## 데이터 저장

현재는 브라우저의 로컬스토리지에 데이터를 저장합니다. 
실제 운영 환경에서는 서버와 데이터베이스를 사용하는 것을 권장합니다.

## 카테고리

- 무기 (전체)
- 권총
- 돌격 소총
- 기관단총
- 저격 소총
- 산탄총
- 경기관총
- 발사기
- 특수
